🔴 PAPER TRADING (виртуальная торговля)

[TOOL:CLOSE_ALL]Закрыть все позиции
[TOOL:PAUSE:true/false]Пауза/запуск сканера
⚙️ СТРАТЕГИЯ
[TOOL:STRATEGY]Текущие параметры стратегии
[TOOL:STRATEGY:SET:sl:3.5]Изменить Stop Loss %
[TOOL:STRATEGY:SET:tp:7.0]Изменить Take Profit %
[TOOL:STRATEGY:SET:trailing_activation:1.0]Трейлинг активация %
[TOOL:STRATEGY:SET:trailing_distance:0.25]Трейлинг дистанция %
[TOOL:SET_PARAM:btc_filter:false]Выключить BTC фильтр
[TOOL:SET_PARAM:recheck:true]⏰ Вкл/выкл фильтр опоздавших входов
📈 ОПТИМИЗАЦИЯ
[TOOL:GRID_OPTIMIZE:30:100]Grid Search (дни:топ_N монет)
[TOOL:APPLY_BEST]Применить лучшие параметры
[TOOL:WFA:30:100]Walk-Forward Analysis
[TOOL:OPTIMIZE_BTC:30:50]🎯 Оптимизация BTC уровней (дни:монет)
[TOOL:OPTIMIZE_BTC:14:100:full]Полная оптимизация (больше комбинаций)
[TOOL:APPLY_BTC]Применить найденные BTC уровни
🧪 БЭКТЕСТ
[TOOL:BACKTEST:current:30:500]Бэктест текущей стратегии (дни:монет)
[TOOL:BACKTEST_PATTERNS:60:200]Бэктест с анализом паттернов
[TOOL:ANALYZE_PATTERNS:30]Анализ реальных сделок
📊 АНАЛИТИКА
[TOOL:MARKET]Обзор рынка и топ пампов
[TOOL:STATS:30]Статистика за N дней
[TOOL:TOP:50]Топ N монет по объёму
[TOOL:ANALYZE_LATE]Анализ опоздавших входов (после сброса PnL)
[TOOL:ANALYZE_LATE:all]Анализ опоздавших (все сделки) — WR/PF свежих vs опоздавших
[TOOL:BACKTEST_CHANGE:30:100]🎯 Найти оптимальный порог входа 5-25% (дни:монет)
📉 ПАМЯТЬ И BLACKLIST
[TOOL:BLACKLIST:DOGE:lowWR]Добавить в чёрный список
[TOOL:BLACKLIST_WORST]Заблокировать худшие из бэктеста
[TOOL:WHITELIST_PROFITABLE]Только прибыльные монеты
[TOOL:MEMORY]Показать память агента

💡 ПОДСКАЗКИ
• Можно просто писать своими словами: «покажи позиции», «запусти бэктест на 30 дней»
• Агент понимает русский и английский
• Для оптимизации рекомендуется 30+ дней и 100+ монет
• OPTIMIZE_BTC подбирает пороги открытия/закрытия по тренду BTC
• Бэктест теперь учитывает BTC автозакрытие (LONG/SHORT при ослаблении тренда)
• ANALYZE_LATE показывает статистику по входам с опозданием (монета вернулась до открытия)
• LIVE команды требуют настроенных API ключей Binance
